gform admin

Manage data

gform admin provides a generic ui to manage data. The connected services (datastores) are defined in app/services.json. The following stores are connected by default.

  • An empty local storage for Bloposts and User
  • An example mongoose application. See installation instructions on github

Swagger client

gform admin is also a swagger client. It is connected to the example swagger application by wordnik

for more information visit admin gform on github
